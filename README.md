# Blue Star Limo Website

This project is a multi-page website for a limo business called **Blue Star Limo LLC**. 

The site is designed to showcase services, introduce the team, and allow users to contact the business for booking inquiries.

## Project Purpose
This website is being developed as a semester-long project for a course. The goal is to gradually build and improve a realistic business website while learning web development concepts.
